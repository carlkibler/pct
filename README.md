In the summer of 2018 I'm hiking [PCT](https://pcta.org/), a ~2600 trail from Mexico to Canada through California, Oregon, and Washington. This is a tracking site / journal thing I built. The design is by [Greg Newman](https://www.gregnewman.org/).

See https://pct.jacobian.org/ for the live site.

**This is not open source.** You're free to read and take inspiration, but please don't re-use without permission. If you want to use this for your own site, get in touch. I'm sure we can work something out.
